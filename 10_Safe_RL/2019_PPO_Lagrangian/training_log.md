# PPO-Lagrangian 訓練日誌

> 環境：SafePendulum（Pendulum-v1 + 代價信號）  
> 安全約束：|θ̇| > 2.0 rad/s → cost=1，每集限 25 步違規  
> 訓練規模：300K 步，n_steps=2048（~146 次更新）

---

## 演算法特有指標

| 指標 | 說明 |
|---|---|
| `eval/mean_reward` | 評估時每集平均獎勵（越高越好，目標 ≥ -300） |
| `eval/mean_cost` | 評估時每集平均違規步數（越低越好，目標 ≤ 25） |
| `eval/lambda` | 拉格朗日乘數 λ（反映安全約束的緊張程度） |
| `train/lambda` | 訓練時 λ 的即時值 |
| `train/ep_cost` | 訓練中每集的違規步數 |

## 對比：PPO-Lagrangian vs CPO

| 項目 | PPO-Lagrangian | CPO |
|---|---|---|
| 更新機制 | 梯度剪裁 + λ 軟約束 | 自然梯度 + 硬約束 |
| 每集代價（最終） | 待填 | 待填 |
| 最終獎勵 | 待填 | 待填 |
| 訓練速度 | 較快 | 較慢（自然梯度計算） |
| 約束保證 | 訓練後期才收斂 | 每步理論上滿足 |
