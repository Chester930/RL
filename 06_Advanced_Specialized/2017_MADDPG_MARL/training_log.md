## 觀察

- **初始期（ep 1–600）**：從 -36.58 快速改善至 -10.78，緩衝區填充後集中式 Critic 開始發揮作用。
- **中期（ep 600–20000）**：均值穩定在 -8 ~ -10，出現週期性峰值（-3 ~ -5）。MADDPG 的高方差源於合作協調的難度與每步更新引入的噪聲。
- **後期（ep 20000–50000）**：峰值逐漸改善，最終在 ep 41800 達全程最佳 -2.04。後 10k 均值（-8.2）略優於前 10k（-8.5），顯示緩慢但持續的學習信號。
- **整體特徵**：SimpleCoopEnv 以每步懲罰狀態範數，最優策略需兩個 agent 持續協調推動狀態趨近原點。高震盪反映協作政策的不穩定，但極值的改善（-3.77→-2.04）確認多 agent 學習有效果。

## MADDPG 核心架構（CTDE）

```
訓練時（集中式）：
  Critic_i 輸入：[obs_1, obs_2, act_1, act_2] → Q_i

執行時（分散式）：
  Actor_i 輸入：obs_i → act_i（不需要其他 agent 資訊）
```

**與 DDPG 的差異**：每個 agent 的 Critic 在訓練時可觀察全域狀態，避免非穩態問題（non-stationarity），是多 agent 協作的關鍵創新。

## 改進建議

- 環境換用 PettingZoo `simple_spread_v3`（3 agents，更接近原始論文設定）
- 每集結束後批次更新（而非每步）以降低梯度噪聲
- 提高 batch_size 或使用優先重播緩衝（PER）改善樣本效率
