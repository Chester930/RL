## 核心概念

### Rainbow = 6 項改進的整合

```
改進項              貢獻
──────────────────────────────────────────────────────────────
1. Double DQN    解耦選擇（線上網路）與評估（目標網路）→ 消除高估偏差
2. PER           按 |TD 誤差|^α 優先取樣 → 樣本效率提升
3. Dueling       Q = V(s) + A(s,a) - mean(A) → 狀態價值更高效更新
4. N-步回報      G = r + γr' + γ²r'' + γ³V(s₃) → 更遠的信用分配
5. C51（分散式） 學習回報的完整分佈，而非純量期望值
6. NoisyNet      可學習雜訊 σ 取代 ε-greedy → 更智慧的探索
```

### C51 分散式 RL：學習回報分佈

```
傳統 DQN：Q(s,a) = E[G_t | s_t=s, a_t=a]    ← 純量期望值

C51：     Z(s,a) = 在 {z_1,...,z_51} 上的機率分佈
          z_i = v_min + (i-1) × Δz          Δz = 10.0000
          z_1 = 0.0（最小回報）, z_51 = 500.0（最大回報）

         優點：保留回報的不確定性資訊（方差、多峰分佈等）
         學習目標：最小化投影後的分佈與目標分佈的 KL 散度
```

### NoisyNet：引數化探索

```
普通線性層：y = Wx + b                  ← 確定性
NoisyLinear：y = (μ_w + σ_w ⊙ ε_w)x + (μ_b + σ_b ⊙ ε_b)

μ：可學習均值（策略主體）
σ：可學習雜訊強度（自動調整探索力度）
ε：每次前向傳遞重新取樣的隨機雜訊

好處：
  - 探索隨狀態而異（比 ε-greedy 更精細）
  - 不需要手動設計 ε 衰減排程
  - 訓練後 σ 自動趨近 0（探索自然消退）
```

### 貝爾曼分佈投影（C51 目標計算）

```
1. 線上網路選最佳動作（Double DQN）：
   a* = argmax_a E[Z_online(s', a)]

2. 目標網路取分佈（N-步折扣）：
   p(z_j | s', a*) from target net

3. 將目標原子投影至支撐集：
   z'_j = clip(r + γ^n * z_j, v_min, v_max)
   分配機率質量到相鄰原子

4. 最小化交叉熵（線上分佈 vs 投影目標）：
   L = -Σ_j m_j * log p(z_j | s, a)
   （使用 PER IS 權重加權）
```

## Rainbow 核心機制詳解

> 以下說明 Rainbow 三個最關鍵改進（NoisyNet / N-步回報 / C51 投影）的計算細節

---

### 一、NoisyNet：可學習的探索強度

```
普通線性層：y = Wx + b                         ← 固定參數，確定性輸出

NoisyLinear：y = (μ_w + σ_w ⊙ ε_w)x + (μ_b + σ_b ⊙ ε_b)
               ↑ 可學習均值    ↑ 可學習雜訊強度  ↑ 隨機取樣雜訊

參數：
  μ_w（可學習）：策略的主體方向
  σ_w（可學習）：探索的強度，初始化為 σ_0 / √fan_in（本訓練約 0.017）
  ε_w（每次前向取樣）：factorised Gaussian，降低計算成本

訓練前：σ_w ≈ 0.017（初始，探索強）
訓練後：σ_w → 0（自動收斂，探索自然消退）
```

**對比 ε-greedy：**

```
ε-greedy（固定排程）：        NoisyNet（自動適應）：
  ε = 1.0 → 0.01（線性衰減）    σ 從初始值自動學習到 0
  所有狀態同樣探索力度           不同狀態的雜訊強度不同
  需手動設計衰減排程             不需要超參調整
```

**本訓練的探索標誌**：無 ε，緩衝區填充（前 2,000 步）由 NoisyNet 的初始雜訊提供探索。

---

### 二、N-步回報（n=3）：更遠的信用分配

```
標準 TD(1) 目標（Double DQN 使用）：
  G_t^{(1)} = r_{t+1} + γ · Q_target(s_{t+1}, a*)     ← 1 步 bootstrap

N-步回報（n=3）：
  G_t^{(3)} = r_{t+1} + γ · r_{t+2} + γ² · r_{t+3} + γ³ · Q_target(s_{t+3}, a*)

本訓練（γ=0.99, n=3）的具體計算：
  假設某 3 步：r_{t+1}=1, r_{t+2}=1, r_{t+3}=1, Q(s_{t+3})=245

  G_t^{(3)} = 1 + 0.99×1 + 0.99²×1 + 0.99³×245
             = 1 + 0.99 + 0.9801 + 0.9703×245
             = 1 + 0.99 + 0.98 + 237.7
             = 240.67

  對比 TD(1)：G_t^{(1)} = 1 + 0.99×245 = 243.55

→ N-步的優點：信用傳播更遠（3步內的真實獎勵，而非純 bootstrap）
→ N-步的限制：需要連續 n 步都是同一策略（off-policy 時有偏差）
→ Rainbow 用 PER 補償：高 TD 誤差的樣本更容易被選中修正
```

### 三、C51 分佈投影（Bellman 目標計算）

C51 不學習純量 Q 值，而是學習回報的**機率分佈**。

**支撐集設定（本訓練）：**

```
n_atoms = 51，v_min = 0.0，v_max = 500.0，Δz = 10.0

51 個原子位置：z_0=0, z_1=10, z_2=20, ..., z_50=500
初始分佈：均勻（各原子機率 = 1/51 ≈ 0.0196）
初始 loss：−log(1/51) = log(51) ≈ 3.93（與訓練日誌記錄的 3.872 相符）
```

**Bellman 分佈投影（每次更新的核心計算）：**

```
以某一樣本（r=1，非終止，γ=0.99，n=3）為例：

1. 目標網路輸出 s' 的分佈：{p_0, p_1, ..., p_50}（51 個機率值）

2. 折扣後移動每個原子位置：
   z'_i = clip(r + γ^3 · z_i, v_min, v_max)
   z'_0 = clip(1 + 0.9703×0,   0, 500) = 1.0   → 落在 (z_0=0, z_1=10) 之間
   z'_1 = clip(1 + 0.9703×10,  0, 500) = 10.70  → 落在 (z_1=10, z_2=20) 之間
   z'_25= clip(1 + 0.9703×250, 0, 500) = 243.6  → 落在 (z_24=240, z_25=250) 之間
   z'_50= clip(1 + 0.9703×500, 0, 500) = 500.0  → 截斷至 v_max

3. 將每個原子的機率質量分配給相鄰的整數原子（線性插值）：
   z'_0=1.0 落在 [z_0=0, z_1=10]，向 z_0 分配 (10-1)/10=0.9，向 z_1 分配 0.1

4. 最小化 KL 散度（交叉熵）：
   L = −Σ_i m_i · log p(z_i | s, a)
   （m_i 是投影後的目標機率，p(z_i|s,a) 是線上網路的輸出）
```

**C51 的優勢（對比純量 DQN）：**

```
純量 DQN：Q(s,a) = 230（點估計，丟失不確定性）

C51：Z(s,a) = 分佈，例如：
  - 60% 機率回報在 [200, 300]（穩定直立期）
  - 30% 機率回報在 [400, 500]（接近滿分）
  - 10% 機率回報在 [0, 100]（早期跌倒）

保留多峰分佈資訊 → 更好的策略評估 → 更穩定的學習訊號
```

### 四、訓練收斂分析

| 里程碑 | loss（交叉熵）| mean Q | 說明 |
|---|---|---|---|
| 首次更新（2K 步）| 3.872 | 251.5 | ≈log(51)，均勻分佈基線 |
| 50K 步 | 2.540 | 131.2 | 快速收斂，KL 下降 34% |
| 100K 步 | 1.797 | 110.3 | 接近滿分，eval=492.3 |
| 150K 步 | 1.975 | 101.2 | 穩定滿分，eval=499.5 |

**loss 在 100K→150K 微升（1.797→1.975）的原因：**

```
分佈越集中 → KL 散度對小誤差更敏感
策略穩定後，分佈縮窄至 z_48~z_50（回報≈480-500）
任何細微偏差的交叉熵懲罰更大
→ loss 微升 ≠ 策略退化（eval 仍 499.5）
```
